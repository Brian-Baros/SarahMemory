===============================================================================
TROUBLESHOOTING
===============================================================================

ISSUE: "TTS engine init failed"
SOLUTION: This is normal on headless servers. Voice synthesis will be
         disabled but all other features will work. Install pyttsx3 for
         offline TTS or gTTS for online TTS.

ISSUE: Progress bars show strange characters
SOLUTION: Your terminal doesn't support Unicode. The system will automatically
         fallback to ASCII characters.

ISSUE: "Datasets directory not found"
SOLUTION: Run SarahMemoryGlobals.py first to create all required directories:
         python SarahMemoryGlobals.py

ISSUE: API server doesn't start
SOLUTION: Check if port 8765 is available. You can change the port in
         SarahMemoryGlobals.py (DEFAULT_PORT variable).

ISSUE: Voice settings not loading
SOLUTION: Check that data/settings/settings.json exists and is valid JSON.
         Delete it to reset to defaults if corrupted.

KNOWN ISSUES & LIMITATIONS
